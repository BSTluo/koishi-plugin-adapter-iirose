### 9.2 验证连接

![连接成功](./../../public/assets/start/28597275-0eb9-4c13-aebd-6b4fb83f9a13.png)

:::warning 连接失败？
如果机器人没有出现在房间中，请检查：
- 优先**查看日志**
- **账户密码**是否正确
- 网络连接是否正常
:::
