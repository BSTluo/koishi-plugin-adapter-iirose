# koishi-plugin-filemanager

## 重要
:::danger
机器人以及各类脚本 需要将富媒体内容上传到第三方图床 以减少蔷薇的服务器的访问压力。

:::
**`filemanager`用于提供图床支持，本页内容将说明如何配置`filemanager`插件。**
